# Facilio Design System

Design system for facilio

## Install

```bash
npm i --save @facilio/design-system
```

```shell
npm i @facilio/design-system
```

# Usage

```jsx
<template>
  <Button appearance="primary">Click Me</Button>
</template>

<script>
import { Button } from "@facilio/design-system";
export default {
  components: { Button },
};
</script>
```
