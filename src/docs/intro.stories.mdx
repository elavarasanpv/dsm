# Theming

## Configuring Theme

<p>
  You can conifugre / override themes in facilio design system by using
  `setTheme` method
</p>

```js
import { setTheme } from '@facilio/design-system'

setTheme({
  type: 'default',
  config: {
    typography: {
      appearances: {
        heading: {
          fontSize: '50px',
        },
      },
    },
  },
})
```

The above example selects the default theme, and overrides the heading typography to have a font size of `50px`.
